What is Ajax


As defined in Wikipedia


AJAX is a group of interrelated web development techniques used to create interactive web applications or rich internet applications. With Ajax, web applications can retrieve data from the server asynchronously in the background without interfering with the display and behavior of the existing page.

Brief History

Year Event
1995 Java applets were introduced - the first way to load data from client side asynchronously
1996 IFrame element were introduced, which also have this functionality
1999 Microsorf created the XMLHTTP ActiveX control in INternet Explorer 5 which when followed by other browsers as XMLHTTPRequest object
2005 The term "Ajax" was borned
2006 The W3C, in order to to make it official web standard, publicized the first draft specification for XMLHTTPRequest object
